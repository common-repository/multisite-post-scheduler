== Description ==

The plugin allows to schedule or publish large number of posts at once for wordpress multisite installation. For each post a user can specify title,content,tags,blog,category,author and publish date.

Main features:

* Quickly schedule or publish large number of posts for multiple sites.

== Installation ==

1. Upload plugin to wordpress plugins directory.
2. Go to plugins section of the dashboard and activate it.

== Frequently Asked Questions ==

= How is the plugin used? =

After activation go to Post scheduler menu and add posts.
